# GNDEC Labs

A collection of lab practicals and code for various subjects from Guru Nanak Dev Engineering College (GNDEC).

## Folder Structure

| Folder Name     | Description                                       |
| --------------- | ------------------------------------------------- |
| `GNDEC-DBMS`    | C++ programs related to Database Management Systems. |
| `GNDEC-DCLD`    | C++ programs related to Digital Circuits and Logic Design. |
| `GNDEC-DS`      | C++ programs for various data structures and algorithms. |
| `GNDEC-OOPS`    | C++ programs demonstrating Object-Oriented Programming concepts. |
| `GNDEC-PAS`     | Python programs for Probability and Statistics.   |
| `Practicals`    | Contains practical files for various subjects.    |
| `Extras`        | Contains extra files and assignments.             |

## How to Use

### C++

To compile and run the C++ programs, you will need a C++ compiler such as g++.

```bash
g++ <filename>.cpp -o <output_filename>
./<output_filename>
```

### Python

To run the Python programs, you will need Python 3 installed.

```bash
python3 <filename>.py
```
